# Mahjong Vision

2023-2학기 모바일앱프로그래밍1 팀프로젝트로 만들었던 앱을 ios, android 하이브리드 앱으로 다시 만들어서 런칭까지 해보는 걸 목표로 한 프로젝트입니다.

마작 패를 카메라로 찍어서 입력하면 점수를 계산해서 보여줍니다.

# 개발 환경설정

react native의 개발 환경설정을 따라갑니다.

## 실행

```
yarn ios
yarn android
```
